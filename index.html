Tentang Saya

Halo! Saya adalah EGI Dian Adi Pradana, mahasiswa aktif di Universitas Amikom. Saya memiliki minat mendalam dalam pengembangan teknologi, khususnya di bidang web development, kecerdasan buatan, keamanan siber, dan dukungan IT. Sebagai mahasiswa, saya terus belajar dan mengembangkan keterampilan praktis melalui proyek-proyek akademik dan pribadi.

Saya percaya bahwa teknologi dapat memecahkan masalah dunia nyata, dan saya berkomitmen untuk berkontribusi melalui inovasi. Di luar akademik, saya suka berkolaborasi dalam tim, berpikir kritis, dan terus meningkatkan diri. Soft skill saya meliputi kemampuan bekerja sama dalam tim, komunikasi efektif, manajemen waktu, dan kemauan belajar hal baru.
